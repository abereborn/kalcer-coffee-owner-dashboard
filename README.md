# 📊 Kalcer Coffee - Owner Dashboard

## 📖 Description
This project is a dashboard interface for the coffee shop owner to monitor business activities.

## ✨ Features
- Dashboard overview
- Employee data
- Attendance tracking
- Sales report
- Stock monitoring

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
